## App Review Notes (v1.0.1)

- Camera permission is used only when the user starts business card scanning to create customer and contact records.
- Photo Library permission is used only when the user selects existing business card or document images to attach to customer and activity records.
- Microphone and Speech Recognition permissions are used only for voice search and voice input in CRM search fields and form fields (for customer, product, and document names).
- Location (When In Use) permission is used only to show nearby customers and places on map screens and to support route planning.
- The app does not use background location tracking.
